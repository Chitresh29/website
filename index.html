Products

Glyfix


GlyphX based on ICR (Intelligent Character Recognition) and Artificial Intelligence technology to convert physical handwritten text into digital format through machine reading. Using a scanner, the hand-written data is read by a software, hence eliminating the need of Manual Data Entry. The focus of the product is a high level of accuracy, with an emphasis on minimizing the cost.

The solution allows data processing in near- Real time. The ability to scale instantly allows processing the peak volumes without any delay.


Boxcite


Boxcite is an integrated marketing and sales platform to record, organize, action, analyze and report customer data to enable enterprises to drive profits from multi-channel marketing programs. Boxcite is built using open source technologies and integrates easily with various CRM, ERP and legacy systems. The applications of Boxcite are several – as a data mart integrating data from multiple sources, for multi-channel demand generation, sales automation, CRM, performance analysis and campaign management.

Charging Gateway

The Charging Gateway implements the Charging Gateway Function on the core network as per the required 3Gpp standards. This is a standard compliant gateway which provides support to all CDRs which standard and manufacturer defined fields.


CDR (Call Data Records) Storage & Retrieval

CDR Storage and Retrieval allows storage of CDRs for any duration for durable storage and retrieval. All CDRs are indexed for instant retrieval based on queries which can be made through the command line or using APIs.
